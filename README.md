# 微信公众号文章批量下载器 (Manifest V3 Chrome Extension)

这是一个基于 Chrome Extension Manifest V3 规范开发的浏览器插件，专为微信公众号创作者或读者设计，用于将**已发表的文章**批量下载到本地。

## ✨ 功能特点

1. **已发表列表深度集成**：在微信公众平台后台（`mp.weixin.qq.com`）已发表文章列表，自动注入多选框和「批量下载」控制面板。
2. **多格式导出支持**：
   - **HTML + 本地图片 ZIP (推荐)**：每篇文章打包为一个 `.zip`，解压后包含本地化的图片文件夹和离线 HTML，完美的离线阅读体验。
   - **Markdown + 本地图片 ZIP**：每篇文章打包为一个 `.zip`，包含 Markdown 文档及关联的本地图片，非常适合导入 Notion, Obsidian 等笔记软件。
   - **单文件 HTML (图片 Base64)**：所有图片均转换为 Base64 编码嵌入在单个 HTML 中，直接双击即可流畅阅读。
   - **纯文本 TXT**：去除所有图片和 HTML 样式，仅保存文章纯文本文字（开头自动整理标题、公众号名称、发表日期与原文链接），文件体积极小，适合纯文字阅读。
3. **防封锁与安全防范**：
   - 支持并发下载控制。
   - 提供**随机延时请求机制**（可在 3~120秒 范围内自定义），模拟人手操作，避免频繁抓取被微信判定为爬虫。
4. **实时状态看板**：
   - 微信后台页面包含一个常驻的进度指示条，展示抓取进度。
   - 插件 Popup 弹窗提供下载队列管理、开始/暂停/清除控制、以及带级别色彩的控制台实时日志。
5. **精美视觉设计**：基于现代微卡片与微信经典绿主题进行视觉定制，极具质感。

## 🚀 安装使用指南

### 第一步：加载插件到谷歌浏览器

1. 打开 Google Chrome 浏览器。
2. 在地址栏输入 `chrome://extensions/` 并回车，进入**扩展程序**管理页面。
3. 开启右上角的 **"开发者模式" (Developer mode)** 开关。
4. 点击左上角的 **"加载已解压的扩展程序" (Load unpacked)** 按钮。
5. 在弹出的文件夹选择框中，选择本项目所在的目录：`D:\code\wechat-download\`。
6. 加载成功后，您会在列表中看到“微信公众号文章批量下载器”。

### 第二步：如何批量导出文章

1. 登录您的 **[微信公众平台](https://mp.weixin.qq.com)**。
2. 点击左侧导航栏的 **「发表记录」** 或 **「已发表」**，进入已群发文章列表页面。
3. 页面载入后，您会发现：
   - 页面右下角自动升起一个 **「微信文章下载助手」** 悬浮控制面板。
   - 每一篇已群发的文章标题左侧，都出现了一个专属的复选框。
4. **勾选您想要导出的文章**（您也可以点击悬浮窗中的「全选本页」）。
5. 点击悬浮窗中的 **「批量下载 (已选数量)」**。
6. 此时任务已发送至后台。您可以：
   - 直接在页面悬浮窗查看实时下载进度。
   - 点击浏览器右上角插件图标（建议将其 pin 在工具栏），打开 **Popup 控制中心**。在 Popup 中，您可以切换**保存格式**（默认 HTML ZIP）、配置**抓取间隔**，并可以在“运行日志”标签页看到详细的防封锁延时、图片解析成功率等日志。
   - 在 Popup 页面中点击 **「开始下载」**，程序即会开始排队抓取。

## ⚠️ 避坑与注意事项

1. **多重保存提示问题**：
   - 如果您的谷歌浏览器设置中开启了 *"下载前询问每个文件的保存位置"*。在批量下载时，浏览器会针对**每个 ZIP 或 HTML 文件**弹窗询问保存路径。
   - **推荐关闭该设置**：建议临时在 Chrome 设置中关闭该选项（进入 `chrome://settings/downloads` -> 关闭「下载前询问每个文件的保存位置」），从而实现静默批量下载。
2. **账号防封建议**：
   - 微信对高频的爬取有非常严苛的限制。批量下载时建议将**抓取间隔**设置为 `3` 至 `8` 秒或更高，限制线程为单线程（默认已强制单线程队列下载），从而保证安全。
   - 抓取过程中若微信官方提示需要进行安全验证，请在浏览器中手动完成验证，并在 Popup 中点击“开始下载”继续。
3. **下载目录路径**：
   - 受 Chrome 插件安全限制，所有导出的文件均会保存在您的系统“下载 (Downloads)”文件夹中，文件格式为 `下载/公众号名称-文章标题-发表日期.zip` 或 `.html`。
