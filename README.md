# 微信/小红书下载器 (Manifest V3 Chrome Extension)

这是一个基于 Chrome Extension Manifest V3 规范开发的浏览器插件，专为**微信公众号创作者/读者**及**小红书创作者**设计，支持一键批量导出微信公众号群发的历史文章和小红书创作者后台的已发布笔记。

---

## ✨ 功能特点

### 1. 微信公众号批量下载 🟢
- **列表深度集成**：在微信公众平台后台（`mp.weixin.qq.com`）已群发列表中自动注入多选框和下载悬浮窗。
- **多格式导出**：
  - **HTML + 本地图片 ZIP (推荐)**：解压后包含本地化图片文件夹和离线 HTML，支持完全离线阅读。
  - **Markdown + 本地图片 ZIP**：适合导入 Notion、Obsidian 等笔记管理软件。
  - **单文件 HTML (图片 Base64)**：所有图片转为 Base64 嵌入在 HTML 中，双击即可无网流畅阅读。
  - **纯文本 TXT**：自动提炼标题、作者、发表日期、原文链接并去除格式，体极小。

### 2. 小红书笔记批量下载 🔴
- **创作者中心深度集成**：在小红书创作者服务平台（`creator.xiaohongshu.com/new/note-manager`）的笔记管理页面自动注入复选框及浮动批量下载条。
- **完美越过安全反爬 (xsec_token)**：基于主页面（Main World）无感网络拦截技术，动态获取每篇笔记的内部 `xsec_token` 校验值，完全绕过小红书新版 Web 端“你访问的页面不见了”防爬锁。
- **高清原图与视频下载**：自动抓取多图笔记的全部无水印超清大图，或是视频笔记的无水印 MP4 文件。
- **无感仿真浏览器渲染**：支持静默加载，确保导出含有最完整的文章正文和发布日期。

### 3. 通用功能 ⚙️
- **防封锁延时机制**：支持可在 3~120 秒范围内自定义的随机延时，模拟真人行为，防止触发平台风控。
- **全局下载队列**：合并下载任务，支持在 Popup 弹窗中进行开始、暂停、清空全局队列的操作。
- **控制台实时日志**：Popup 弹窗包含实时带颜色日志看板，随时知晓下载任务的执行状态。

## 🚀 安装使用指南

### 第一步：加载插件到浏览器

1. 打开 Google Chrome 浏览器。
2. 在地址栏输入 `chrome://extensions/` 并回车，进入**扩展程序**管理页面。
3. 开启右上角的 **"开发者模式" (Developer mode)** 开关。
4. 点击左上角的 **"加载已解压的扩展程序" (Load unpacked)** 按钮。
5. 选择本项目所在的目录 `wechat-download`。
6. 载入成功后，列表中会显示 **“微信/小红书下载器”** 及精美的融合 3D 图标。

### 第二步：如何批量导出内容

#### 1. 导出微信公众号文章
1. 登录 [微信公众平台](https://mp.weixin.qq.com)。
2. 点击左侧导航栏的 **「发表记录」** 或 **「已发表」**，进入群发文章页面。
3. 页面载入后，右下角会自动升起悬浮窗，勾选文章，点击 **「批量下载」** 即可。

#### 2. 导出小红书笔记
1. 登录 [小红书创作者服务平台](https://creator.xiaohongshu.com/)。
2. 点击左侧侧边栏的 **「笔记管理」** 进入已发布笔记列表。
3. 页面完全加载后，右下角会展现浮动的红色下载控制台。
4. **勾选您想要导出的笔记**（支持全选本页），点击 **「批量下载」** 提交任务，后台仿真引擎会自动执行抓取与保存。

## ⚠️ 避坑与注意事项

1. **临时关闭“下载前询问”**：
   - 建议在 Chrome 设置中进入 `chrome://settings/downloads` -> 临时关闭 **「下载前询问每个文件的保存位置」**，否则批量下载时浏览器会针对每个 ZIP 或 HTML 密集弹窗询问保存路径。
2. **下载目录**：
   - 受 Chrome 插件安全限制，所有文件默认均会保存在您操作系统的 `下载 (Downloads)` 文件夹中。
